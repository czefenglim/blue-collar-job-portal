# Activity Diagrams

This folder includes PlantUML activity diagrams covering the main flows for each use case.

## Common Use Cases
- `docs/activity-auth.puml` — Sign Up / Login
- `docs/activity-chat.puml` — Chat
- `docs/activity-notify.puml` — Notifications
- `docs/activity-prefs.puml` — Preferences

## Render Instructions
- VS Code: install "PlantUML", open any `.puml` and preview.
- Online: paste contents into https://www.plantuml.com/plantuml/.
- CLI: `java -jar plantuml.jar docs/activity-auth.puml` (repeat for others).
